# 01 - Introduction to Front-End Development

This is the first course in the Meta Front-End Developer program.

The goal of this course is to provide an introduction to front-end web development, featuring the daily responsibilities of front-end developers and the essential technologies that enable the web. 
This includes **HTML** and **CSS**, as well as modern UI frameworks such as **Bootstrap** and **React** for interactive and responsive user experiences.

By the end of the course, it is expected to be able to:  
- Describe the front-end developer role. 
- Explain the core and underlying technologies that power the internet. 
- Use HTML to create a simple webpage.
- Use CSS to control the appearance of a simple webpage. 
- Explain what React is.
- Describe the applications and characteristics of the most popular UI frameworks.

The final project focuses on building a **responsive webpage** with HTML and the Bootstrap CSS framework that will change layout depending on the size of the screen viewing it.
