## Introduction
This lab introduces the command line as an essential developer tool and focuses on identifying the current working directory. Understanding where you are in the file system is fundamental when navigating projects and executing commands.

## Goal
The goal of this lab is to learn how to use the `pwd` (Print Working Directory) command to display and understand the current working directory in the terminal.

## Objectives
- Open and use the terminal interface.
- Execute the `pwd` command correctly.
- Interpret the output of the `pwd` command.
- Understand the importance of the working directory when using relative paths.

## Instructions
1. Open a new terminal window.
2. Type the `pwd` command.
3. Press Enter to execute the command.
4. Observe the output path displayed in the terminal.
5. Understand that this path represents the current working directory and serves as the reference point for file operations.

## Key Takeaways
- The `pwd` command prints the current working directory.
- Knowing your working directory is essential for navigating and managing files.
- The command line is case-sensitive, so commands must be typed accurately.
- Understanding the working directory helps prevent mistakes when running commands.
