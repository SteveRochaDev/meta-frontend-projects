# Little Lemon Home Page

## Project Purpose
This is a semantic, responsive home page for the fictional restaurant Little Lemon, built as part of the HTML and CSS in depth course.

## Technologies Used
- HTML5 (semantic elements)
- CSS3 (flexbox and grid)
- Responsive design

## Layout
- **Header:** Logo and navigation menu with four links
- **Main content:** 
  - Row 1: Full-width promotional banner
  - Row 2: Three columns with image + text
- **Footer:** Two columns (left: smaller logo, right: copyright text)

## Interactivity
- Hover effects on navigation links and feature articles
- Subtle scaling animation on feature cards
