# 04 - HTML and CSS in Depth

This course is part of the Meta Front-End Developer program.

The focus of this course is on **HTML** and **CSS**, providing a deeper understanding of front-end web development. 
It covers creating responsive layouts, building forms, using **Bootstrap** components, and applying semantic HTML 
to structure web pages effectively. Debugging techniques are also introduced to identify and fix common front-end issues.

The course includes 3 modules with videos, readings, and programming assignments to practice HTML, CSS and Bootstrap concepts.

The final projects involve creating responsive layouts and interfaces while applying debugging tools to ensure robust web page functionality.
