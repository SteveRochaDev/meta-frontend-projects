# Lab: Create a Grid Layout

## Introduction
This lab demonstrates how to use CSS Grid to create a responsive Holy Grail layout. The layout includes a header, footer, main content area, and sidebars. CSS Grid allows for clean and maintainable code for complex page structures.

## Goal
To create a grid layout using CSS Grid properties and grid template areas that adapt to different viewport sizes.

## Objectives
- Define a CSS grid layout using `display: grid`.
- Use `grid-template-columns` and `grid-template-rows` for structuring the layout.
- Assign `grid-template-areas` for semantic placement of sections.
- Implement responsive design using media queries.

## Commands Practiced
- CSS Grid properties: `display: grid`, `grid-template-columns`, `grid-template-rows`, `grid-template-areas`.
- Media queries: `@media` for responsive adjustments.

## Key Takeaways
- CSS Grid simplifies complex layouts without floating or positioning hacks.
- `grid-template-areas` enhances readability and maintainability.
- Media queries allow layouts to adapt to different screen sizes.
