# Lab: Rate the Media

## Introduction
HTML provides native support for embedding media and collecting user input through interactive form controls. Video elements and range inputs allow users to consume content and provide feedback without requiring external libraries or JavaScript.

This lab focuses on creating a video player with a rating form using standard HTML elements.

## Goal
The goal of this lab is to embed a video player and create a rating form that allows users to rate the media on a scale from 1 to 5.

## Objectives
- Embed a video using the HTML `<video>` element.
- Enable playback controls for the video.
- Create a form with a range slider input.
- Use a datalist to display rating values.
- Submit the selected rating using a form button.

## Features Implemented
- Video player with playback controls.
- Range slider limited between values 1 and 5.
- Datalist showing rating steps.
- Submit button to send the selected rating.

## Key Takeaways
- HTML supports native video playback.
- Range inputs are useful for selecting numeric values.
- Datalists enhance usability by showing predefined options.
- Media interaction and user feedback can be implemented without JavaScript.
