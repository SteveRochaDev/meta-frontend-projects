# 08 - Front-End Developer Capstone

This course is part of the Meta Front-End Developer Professional Certificate.

The Capstone project provides a platform to demonstrate skills acquired throughout the certificate program by solving a real-world problem. 
Each module includes a recap of previously covered concepts and links to relevant content.

The course guides through building a complete front-end application using **React JS** and **JavaScript**, 
focusing on creating multiple views with reusable components, styling responsive user interfaces, ensuring clean and bug-free code 
and implementing usability best practices.

On completion, a job-ready portfolio is developed to showcase abilities in front-end web development, 
demonstrating proficiency to recruiters and potential employers. 

The course includes 4 modules with videos, readings, and assignments to practice the concepts covered.
