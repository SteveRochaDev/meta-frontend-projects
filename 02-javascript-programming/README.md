# 02 - Programming with JavaScript

This course is part of the Meta Front-End Developer program.

JavaScript is introduced as the programming language that powers the modern web. 
The course covers functions, objects, arrays, variables, data types, and the HTML DOM. 
It also explores testing JavaScript code using **Jest** and introduces **Node.js** and **npm**, 
including how to install packages and work with `package.json`.

Interactive possibilities with modern JavaScript technologies are demonstrated, 
along with simple unit tests to practice testing concepts.

The course includes 5 modules, with videos, readings, and programming assignments to 
practice the concepts covered.
