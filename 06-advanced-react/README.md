# 06 - Advanced React

This course is part of the Meta Front-End Developer program.

Advanced React concepts are explored, including component composition, advanced hooks, custom hooks 
and patterns like Higher Order Components and Render Props. 
The course also covers testing React applications using React Testing Library and integrating commonly 
used React libraries for more efficient development.

The course includes 4 modules with videos, readings, and programming assignments to practice the concepts. 
Topics include rendering lists and forms efficiently, managing shared state, using React Context, 
fetching data from remote servers and building robust, reusable components.

Projects focus on applying advanced techniques to build scalable, maintainable, and testable React applications, 
including consuming API data and creating portfolio-ready projects.
