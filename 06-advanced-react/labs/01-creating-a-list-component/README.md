# Creating a List Component

## Description
This lab demonstrates how to create a React component that renders a list of items. Specifically, we built a `DessertsList` component to display desserts with less than 500 calories, sorted by calories in ascending order.

## Tasks Completed
- Created a `DessertsList` component
- Filtered desserts with less than 500 calories
- Sorted the desserts by calories in ascending order
- Rendered each dessert as an `<li>` element in the format: `${dessertName} - ${dessertCalories} cal`
- Verified the output in the browser using the development server
