# Dynamic Events

## Description
This lab demonstrates how to handle user-generated events in a React application. A button click triggers a function that generates a random number and interacts with the user through prompts and alerts.

## Tasks Completed
- Added a button element to the App component
- Implemented an event handler function to respond to user clicks
- Generated a random number between 1 and 3
- Used `prompt` to capture user input
- Displayed results using an alert dialog
