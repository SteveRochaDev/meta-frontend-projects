# Song Selection App

## Description
This lab demonstrates how to handle user interactions in a React application using event handlers. Buttons are used to play and pause different audio files based on user input.

## Tasks Completed
- Created buttons to trigger audio playback
- Used the `Audio` object to load sound files
- Implemented toggle logic to play and pause audio
- Attached event handlers using `onClick`
