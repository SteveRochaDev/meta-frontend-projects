# Build a Calculator App

## Description
This lab consists of building a simple calculator application using React. The calculator supports the four basic arithmetic operations and demonstrates the use of `useState` and `useRef` for managing state and accessing DOM elements.

## Tasks Completed
- Implemented addition, subtraction, multiplication, and division operations
- Used `useState` to manage the calculator result
- Used `useRef` to access input and result elements
- Added validation to prevent division by zero
- Implemented reset functionality for both input and result
- Styled the application using a custom `App.css` file
