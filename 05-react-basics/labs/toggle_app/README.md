# Dynamic Component Rendering in React

## Description
This lab demonstrates how to manage state and conditionally render components in a React application. The app switches between logged-in and logged-out states using the `useState` hook.

## Tasks Completed
- Created separate components for Login, Logout, and Home Page
- Used the `useState` hook to manage the logged-in state
- Passed login and logout functions as props to child components
- Implemented conditional rendering based on authentication state
- Organized the application using a component-based structure
