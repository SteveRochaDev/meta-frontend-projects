# React Counter App

## Description
This lab demonstrates how to manage state in a React application using the `useState` hook. A simple counter app is built where users can increase or decrease a number through button click events.

## Tasks Completed
- Initialized state using the `useState` hook
- Displayed the current counter value dynamically in the UI
- Implemented an "Add 1" button to increment the counter
- Implemented a "Subtract 1" button to decrement the counter
- Handled user interactions using `onClick` event handlers
- Applied basic styling using `App.css`
