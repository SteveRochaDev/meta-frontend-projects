# Creating Multiple Card Components

## Description
This lab demonstrates how to create a reusable React component and render it multiple times using props. Each instance of the Card component receives different data from the parent App component.

## Tasks Completed
- Created a new `Card.js` component
- Used `props.h2` and `props.h3` to display dynamic content
- Rendered three `Card` components in `App.js` with different prop values
- Applied CSS styling through the `card` class
