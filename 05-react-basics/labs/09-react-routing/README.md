# Creating a Route

## Description
This lab demonstrates how to implement client-side routing in a React application using React Router. A navigation bar allows switching between different pages without reloading the browser.

## Tasks Completed
- Installed `react-router-dom`
- Set up `BrowserRouter` in `index.js`
- Created a new `Contact` component
- Added a new route for `/contact`
- Added a navigation link to access the Contact page
- Verified routing between Homepage, About, and Contact pages
