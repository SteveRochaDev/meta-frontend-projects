# Displaying Images in React

## Description
This lab demonstrates how to display an image in a React application by importing it from the `assets` folder and rendering it inside a component using JSX.

## Tasks Completed
- Imported an image from the `src/assets` folder
- Rendered the image inside the `App` component
- Used the `img` element with the `src` attribute set to the imported image
- Added an appropriate `alt` attribute for accessibility
