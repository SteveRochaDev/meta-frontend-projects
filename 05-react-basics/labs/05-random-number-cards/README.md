# Random Number Cards

## Description
This lab demonstrates how to create reusable React components, pass data using props, and apply conditional rendering. The app dynamically generates random numbers and displays three Card components with their value and whether the number is "High" or "Low."

## Tasks Completed
- Created a new `Card.js` component  
- Used `props.num` inside the `Card` component to display dynamic numbers  
- Implemented conditional rendering using a ternary operator to show "High" or "Low"  
- Rendered three `Card` components in `App.js` with different random numbers  
- Applied CSS styling through the `card` class
