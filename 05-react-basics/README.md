# 05 - React Basics

This course is part of the Meta Front-End Developer program.

React is introduced as a powerful JavaScript library for building user interfaces in web and mobile applications. 
The course covers fundamental concepts such as reusable components, props, forms, and dynamic rendering. 
It also demonstrates how to organize React projects for scalability and maintainability.

The course includes 4 modules, with videos, readings, and programming assignments to practice React concepts. 
Tools used include React.js, JSX, HTML, CSS, JavaScript, and VSCode.

Final projects focus on creating interactive and dynamic web applications, applying forms and components to manage data and enhance the user experience.
